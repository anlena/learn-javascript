  ```
  2.npm install -g babel-cli
  3.npm install --save-dev babel-preset-es2015 babel-cli
  4.在根目录下建一个文件
  5.内容为{
      "presets": [
          "es2015"
      ],
      "plugins": []
  }
  6.简化转化命令，更改package内容 "scripts": {
          "build": "babel src/index.js -o dist/index.js"
      },

  ```
